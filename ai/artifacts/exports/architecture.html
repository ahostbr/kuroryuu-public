Kuroryuu Architecture Overview

# Kuroryuu Architecture

## Components

- **MCP Core** - Tool server on port 8100
- **Gateway** - FastAPI on port 8200
- **Desktop** - Electron app
- **Web** - Next.js frontend

## Multi-Agent System

1. Leader coordinates workers
2. Workers execute tasks
3. k_inbox for messaging
4. k_pty for terminal control
